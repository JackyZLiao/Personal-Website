SPELL CHECKER

    C PROGRAM

C-based spell-checking tool that leverages hash tables to efficiently identify misspelled words in text files.

Project Background

This project was my first project I did using the language I started with, C. This was a simple but fun project that helped sharpen some of my coding skills and hone in on my knowledge of data structures and algorithms.

It takes in a dictionary, and loads it into a dynamically allocated hash-table. In order to reduce collisions, I used a variation of the "djb2" hash function developed by Daniel J.Bernstein.

The use of a collision-reducing hash function allows the dictionary to load in a 143,091 word dictionary and check a 17,756 word text file in 0.05 seconds.

Languages/Technologies

C

Concepts

Pointers | Hash tables | Hash functions | Dynamic Memory Allocation | Data Structures and Algorithms



Status: Complete
